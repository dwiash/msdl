= msdl

Download manga from mangastream **CURRENTLY BROKEN**

== Requirement

* Ruby (http://ruby-lang.org)
* Rubygems (http://rubygems.org/)
* Nokogiri (http://nokogiri.org/)
* Wget (http://www.gnu.org/software/wget/)

== Usage

    ./msdl.rb [manga URL]

example:

    ./msdl.rb http://mangastream.com/read/one_piece/55133868/1
